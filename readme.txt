Приложение для мониторинга количества звонков менеджеров колл-центра с возможностью установки списка менеджеров. 
Данные приложение получает из Битрикс24 с помощью js библиотеки для работы с bx24 rest api.
Имеет возможность задания плана по звонкам, установки периода и типа звонков(входящие, исходящие).

Для установки необходимо авторизоваться в Битрикс24 с правами администратора
